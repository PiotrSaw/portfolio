PassApp

Password management system

About the system

PassApp was created by a three-person team as part of an engineering project. The goal of the project was to develop a secure password management system that allows users to store and organize their passwords in a secure and convenient way. Users have access to their data through a desktop application and a mobile application for the iOS platform. It is based on zero-knowledge architecture, meaning that even the developers of the application do not have access to stored passwords. It also allows generating strong passwords, generating 2FA (two-factor authentication) codes, and checking for password leaks in the Have I Been Pwned database. The project was implemented in Typescript and Swift using frameworks such as Express, Next.js, React, Electron and SwiftUI. Data is stored in a PostgreSQL relational database.


In this project, I was responsible for designing and implementing the database and REST API server along with integration tests. The server is responsible for user authentication, session management, password data management, and communication with client applications.

Technologies

PassApp Projekt uses this technology stack:

  • Backend: Express, Jest, PostgreSQL, Docker
  • Frontend: React, Next.js, Shadcn, Tailwind CSS, Jest
  • Desktop: Electron
  • Mobile: SwiftUI (iOS)
