Tests generator

Application for teachers

About the system

The application was designed for teachers who want to quickly create tests for their students but prefer to create their own assignments. The system allows them to create assignments divided into their own categories, creating a database of questions from which they can randomly generate tests. The teacher selects the number of groups and then adds questions to the test by selecting a category and clicking "Add." The system then randomly selects questions from that category (the number of questions equals the number of groups) and adds one to each group. If only one group is selected, the teacher can manually add a question to the test by entering its ID. The application supports adding illustrations to each question (although this is not mandatory), and allows the entry of advanced mathematical formulas using the LateX format. Once generated, the test is displayed as a PDF file.

Technologies

The project uses the following technology stack:

  • JavaScript
  • Express
  • EJS view templates
  • MongoDB
