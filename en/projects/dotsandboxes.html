Dots and Boxes

2 players game

About the application

The Dots and Boxes application is a computer adaptation of the classic paper game of connecting dots to create squares. The player scores points for each square he or she closes. The aim of the game is to score more points than your opponent by strategically connecting the dots and blocking your opponent's moves. The user can choose their name, color and board size. The application was created entirely in Java using the javax.swing and java.awt libraries, launched in the IntelliJ IDEA environment.

Technologies

The project uses this technology stack:

  • Java
  • javax.swing
  • java.awt
